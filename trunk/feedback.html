How to Give Feedback on the Book...  

Please enter comments and suggestions about the book or this website into our Bugzilla bug database, project name "producingoss".

You will need to create a Bugzilla account. This only takes a minute; just follow the on-screen instructions. Once you have an account, you can use it for all future feedback — that is, you only create an account once, not every time you want to send in a suggestion.

Why a bug tracker instead of email?

Suggestions often need to be integrated with one another before being implemented, which is much easier if they are recorded in a single database. Also, using a tracker makes it easy for anyone to browse existing comments and suggestions, which enables people to find out if the feedback they were about to send has already been sent by someone else.

If this book were a full-fledged open source project, it would have mailing lists, file-sharing areas, archives, and all the usual stuff. However, it's somewhere between a single-author work and a collaborative project, so its supporting infrastructure is designed to make things easy for the primary maintainer. As a special exception, if you know my personal email address and can't stand the thought of using the bug tracker, then just mail your comments to me, and I'll take it from there.
