Here's a alphabetical list of words that are not easy to translate. Each
paragraph starts with the english term/phrase, then one line per proposed
translation. If possible with origin. Proposals should be sorted by
preffered ussage.

author tag:
Markierung des Authors
Author Markierung 

bug:
Bug
Fehler

bug tracker:
Bug Tracker

brach:
Zweig

code:
Code

deployment:
das Deployment
Installation (only when the context allows)

fix:
Fix (only when you can't circumscribe it
	i.e "einen Fehler beheben")

issue tracker:
Ticket System

major number _1_.2.3:
Hauptversionsnummer

micro number 1.2._3_:
revisionsnummer (not sure about this one)

minor number 1._2_.3:
Nebenversionsnummer

release:
neue Version
Version (when it's clear from the context that it is a new version)
Veröffentlichung (again only when context allows)

revision:
Revision
Version

to build:
einen Build machen
kompilieren (only when correct in context)
to commit:

committen (also dev slang, in our company anyway)
einen Commit machen (alternative noun form)
einchecken (german developer slang)

to merge:
mergen (german developer slang)
einen Merge machen (alternative noun form)
zusammenführen (translation of the subversion book)
mischen (I don't think this is correct, but
	show me the context an I might change my mind)

version control:
Versionsverwaltung
Versionskontrolle (don't like this one personally but it seems
	to be in fairly widespread use)
