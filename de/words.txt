Here's a alphabetical list of words that are not easy to translate. Each
paragraph starts with the english term/phrase, then one line per proposed
translation. If possible with origin. Proposals should be sorted by
preffered ussage.

author tag:

Markierung des Authors
Author Markierung

Bug
Fehler

Bug Tracker

brach:

Code

Installation (only when the context allows)

Fix (only when you can't circumscribe it
	i.e "einen Fehler beheben")

framework:
das Framework

issue tracker:
Ticket System

john random:
Hans Mustermann (Only a placeholder since it doesn't
	capture quite the same meaning)

Mailingliste

lizensfrei
