How to Give Feedback on the Book...  

Feedback / Translation Inquiries

To send suggestions about the book or this website, please contact the author. The same method should be used for translation inquiries.

(If this book were a full-fledged open source project, there would be an open mailing list with web-accessible archives, among other things. However, the book stands somewhere between a single-author work and a collaborative project, and its supporting infrastructure is designed mainly to make things easy for the primary maintainer.)
